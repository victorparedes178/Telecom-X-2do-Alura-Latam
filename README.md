# Telecom-X-2do-Alura-Latam
Proyecto de Alura Latam, un analisis en el cual tuvimos que poner en practica todo lo visto en el bloque. Se utilizo los datos filtrados de la primera base de datos.
Se hicieron los correspondientes modelos, despues de limpiar, normalizar datos. Se hicieron Tablas de correlacion para averiguar cual es el factor con una relacion mas alta. 
Se hicieron 2 modelos, uno de arbol, y uno de correlacion lieal.
